A mortgage comparison form with a shortcode and Elementor widget that calculates monthly payments, total deal costs, and savings between current and new rates.

== Description ==
Use the `[mortgage_rate_compare]` shortcode or the **Mortgage Rate Compare** Elementor widget to display a calculator that compares your current mortgage rate to a new offer. Enter the balance, remaining term, deal period, rates, and optional fees to see payment amounts, total costs, and potential savings.

== Installation ==
1. Download the ZIP or create one locally from this repository root:
   `zip -r mortgage-rate-comparison.zip mortgage-rate-comparison`
2. In WordPress admin, go to **Plugins → Add New → Upload Plugin**.
3. Upload `mortgage-rate-comparison.zip`, then click **Install Now** and **Activate**.
4. Add the shortcode to a page or insert the Elementor widget to customize the design.

== Frequently Asked Questions ==
= Can I customize the design? =
Yes. The plugin includes an Elementor widget with style controls for the container, labels, inputs, buttons, and results. You can also override the provided CSS in your theme or Elementor if needed.
